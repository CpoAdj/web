Información del Proyecto

Descripción del Proyecto

Este proyecto, llamado Tria-Guante GT, se centra en la creación de un guante inteligente capaz de medir y monitorear signos vitales como la temperatura corporal y la frecuencia cardíaca. El objetivo es utilizar esta herramienta en situaciones de emergencia para realizar un triaje rápido y efectivo.

Objetivos

  • Desarrollar un dispositivo portátil y fácil de usar para medir signos vitales.
  • Implementar tecnología de sensores avanzados para obtener datos precisos.
  • Facilitar el proceso de triaje en emergencias médicas.

Metodología

El desarrollo del proyecto siguió una metodología ágil, permitiendo una rápida interación y mejora continua. Se realizaron pruebas exhaustivas para asegurar la precisión de los datos y la usabilidad del dispositivo.
